### 属性

| 属性名  | 类型  | 作用                                         | 默认值 |
| ------- | ----- | -------------------------------------------- | ------ |
| options | Array | 传入的候选项，必须是`[{label: '', code:''}]` | []     |

### 事件

| 事件名   | 回调参数     | 备注                      |
| -------- | ------------ | ------------------------- |
| input    | 选择的 value | 建议通过 v-model 指令绑定 |
| selected | 选择的数据   | 返回点击选择的整条数据    |
